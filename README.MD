# 🛡️ Cognitive Immune System – Cybernetics of Vigilance

**MVP (Minimum Viable Product)**  
Built for the RIFS Fellow Programme 2027  
*"Sustainability in Tough Times"*

---

## 🔍 About

This web tool is the first operational prototype of the **"Cybernetics of Vigilance"** framework.  
It detects **hidden gaps** that erode trust and legitimacy in sustainability governance by analyzing texts
(policy speeches, reports, media).

The system produces:
- A **Diagnostic Card** with scores for rigidity, absence, and hidden pain.
- A **"Forbidden Question"** – a cognitive vaccine against institutional blindness.

It works locally using keyword analysis, and gains deeper insight when connected to the **DeepSeek AI API**.

## 🛠️ How to Use

1. Paste any text (policy, speech, article) into the box.
2. Optionally, enter a free DeepSeek API key for AI-enhanced analysis.
3. Press **"Run Vigilance Diagnosis"**.
4. View the Diagnostic Card and your unique "Forbidden Question".
